Guide to Use OCR Service

This is a guide on how to use the OCR service in our application.

Step 1: Upload an Image

Click on the "Upload" button and select the image you want to analyze. The image should contain clear, readable text.

Step 2: Edit the Image

Use the provided tools to edit the image as needed. You might be able to crop, rotate, or adjust the contrast of the image.

Step 3: Select the Document Language

Select the language of the text in the image. This helps the OCR algorithm accurately recognize the text.

Step 4: Select the Output Format

Select the format you want the extracted text to be in. This could be plain text, a Word document, a PDF, etc.

Step 5: Test the Text Detection Algorithm

Click on the "Test" button to test the text detection algorithm. This will show you where the algorithm detects text in the image.

Step 6: Predict the Text

Click on the "Predict" button to start the OCR process. The application will analyze the image and extract the text.

Step 7: Download the Text

Once the OCR process is complete, click on the "Export" button to download the extracted text.
